div es un contenedor generico. No tiene efecto en el contenido o distribucion hasta que se estilice de alguna manera usando CSS Debera ser ultilizado solo cuando otro elemento semantico(article, nav) no es apropiado
span Es contenedor generico en linea para contendido de redaccion, puede ser usado para agrupar elementos para propositos de estilo, o porque comparten atributos en comun Deberia ser usado solo cuando no otros elementos semanticos no son apropiados.
h1 - h6 Representan seis niveles de encabezados de seccion. h1 es el mas alto nivel y h1 es el menor Informacion de encabezado puede ser usada por diferentes agentes para construir automaticamente una tabla de contenidos para un documento
body Representa el contenido de un documento de HTML. Solamente puede haber una etiqueta body en el documento
table Representa datos tabulados que es informacion representado en una tabla bi-dimensional comprendida de filas y columnas de celdas conteniendo datos Define una tabla de datos
article Representa una composicion autocontenida en un documento, pagina, aplicacion o sitio, el cual se pretende que sea independientemente distribuible o reusable Cada articulo deberia estar identificado, tipicamente, una cabezera h1-h6 como un elemento hijo
